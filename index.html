CSV/Excel Upload Page
Route: /invoices/create/csv

Design Notes

This page allows users to upload a CSV/Excel file and map columns to create invoices directly. Key design features:

  • Simplified single-page workflow that combines file upload and column mapping
  • Clear file upload area with drag-and-drop support
  • Intuitive column mapping interface with required fields clearly marked
  • Sample data display to help users verify their mapping
  • Direct "Create Invoices" button that processes the file immediately

This streamlined approach allows users to quickly create invoices from CSV/Excel files without multiple steps. After creation, users will be directed to the validation page where they can verify and fix any issues with the generated invoices.

Invoice Processing Page
Route: /invoices/{id}/processing

Design Notes

This page provides visual feedback during the automated processing of an invoice after creation. Key features include:

  • Clear loading animation to indicate processing is in progress
  • Step-by-step progress indicators showing the current stage of processing
  • Progress bar to give users a sense of how much longer they need to wait
  • Informative messages explaining what's happening at each step

The page is designed to keep users informed during what might otherwise be perceived as system lag, improving the overall user experience by providing transparency into the background processes.
